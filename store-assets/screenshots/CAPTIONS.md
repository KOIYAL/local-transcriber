# Microsoft Store screenshot captions

各キャプションは200文字以下です。各言語のStore登録情報へ、対応する画像と
同じ順序で登録してください。

## ja-JP

1. `01-ready.png`
   ファイルを選ぶだけ。難しい設定やAPIキーなしで、音声・動画をこのPC上で文字起こしできます。
2. `02-file-selected.png`
   音声・動画ファイルをドラッグ＆ドロップ。PCに適したAIモデルはアプリが自動で選択します。
3. `03-processing.png`
   処理状況をわかりやすく表示。初回セットアップ後はオフラインでも文字起こしできます。
4. `04-result-and-exports.png`
   タイムスタンプ付きの結果を確認し、TXT・SRT・VTT・JSON形式で書き出せます。

## en-US

1. `01-ready.png`
   Select a file and transcribe audio or video locally, with no API key or complicated setup.
2. `02-file-selected.png`
   Drag and drop an audio or video file. The app automatically selects a model suited to your PC.
3. `03-processing.png`
   Follow transcription progress at a glance. After first-time setup, transcription works offline.
4. `04-result-and-exports.png`
   Review timestamped text and export the completed transcript as TXT, SRT, VTT, or JSON.
